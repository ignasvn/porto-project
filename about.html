About the Porto Website

The Porto website is a platform that displays a collection of portfolios from various individuals in one place. The main goal of this website is to make it easy for anyone to showcase their work or experience professionally, while also providing references and inspiration for visitors.

This website was created as part of an assignment for the Web-Based Programming 1 course, focusing on applying HTML and CSS concepts to build attractive static displays. In the Web-Based Programming 2 course, this website will be developed into a dynamic system.
